# Inflation Prediction System

An intelligent machine learning-based system designed to predict inflation trends using historical economic data and key financial indicators.

# Overview

The Inflation Prediction System aims to forecast inflation rates by analyzing economic variables such as GDP, unemployment rate, and other macroeconomic indicators.

Inflation forecasting is a complex problem influenced by multiple dynamic factors, and machine learning models help capture these patterns more effectively than traditional methods .

# Features
* Predict future inflation trends
* Data preprocessing and feature engineering
* Machine Learning model for prediction
* Visualization of inflation trends
*  User-friendly interface (if Streamlit used)
  
# Tech Stack
- Programming Language: Python
- Libraries: Pandas, NumPy, Scikit-learn
- Model: Regression / ML Model (custom)
- Visualization: Matplotlib / Plotly
- Deployment: Streamlit
  
# Setup
1. Installation
   
    git clone https://github.com/D-Bavanapriya/Inflation-prediction-.git

    cd Inflation-prediction-

2. Create virtual environment:

    uv venv
    .venv\Scripts\activate

3. Install dependencies:

    uv pip install -r requirements.txt

4. Run the App

    streamlit run app.py
